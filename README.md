# Fotoblog e Galeria de Imagens

## Apresentação

Bem-vindo ao Fotoblog e Galeria de Imagens! Este repositório destaca as habilidades adquiridas no curso Explorer - Rocketseat, abrangendo `display grid`, animações CSS e integração de API para alimentar o banco de imagens. Explore um mundo visualmente atraente de fotografias e momentos cativantes.

## Sobre o Projeto

O projeto visa criar um fotoblog e galeria de imagens, proporcionando uma experiência visual envolvente. Cada imagem conta uma história, e a aplicação de animações CSS adiciona um toque de dinamismo, tornando a exploração do fotoblog uma jornada única.

## Conteúdo do Projeto

O Fotoblog e Galeria de Imagens incluem:

- **Animações CSS:** Adicionando vida às imagens e criando transições suaves.
- **API de Banco de Imagens:** Integração de uma API para carregar e exibir imagens de forma dinâmica.
